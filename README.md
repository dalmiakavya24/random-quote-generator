# Random Quote Generator

This is a simple Python app that shows a random motivational quote when the user clicks a button.

## Features

* Displays random quotes
* Simple graphical interface using Tkinter
* Easy to use

## Built With

* Python
* Tkinter

## How to Run

1. Download the project
2. Open the folder
3. Run the file:

python app.py

## Example

Click the button and the app will show a random quote each time.
